    Σημασιολογικός και Κοινωνικός Ιστός

Οργάνωση πληροφορίας στον Παγκόσμιο Ιστό: semantics, οντολογίες και γλώσσες σημασιολογίας (RDF, OWL). Η γλώσσα ερωτημάτων SPARQL. Ανοικτά Διασυνδεδεμένα Δεδομένα. Επεξεργασία chat text, text analytics, text mining και web sentiment analysis. Επεξεργασία δεδομένων από κοινωνικά δίκτυα. Τεχνολογίες Web 2.0 και μηχανές αναζήτησης Ιστού.
